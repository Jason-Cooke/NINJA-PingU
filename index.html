What's NINJA PingU?

"NINJA-PingU Is Not Just a Ping Utility" is a free open-source high performance network scanner and services discoverage tool for large scale analyses. It has been designed with performance as its primary goal and developed as a framework to allow easy plugin creation.


Why's NINJA PingU?

NINJA PingU is open source, it has been designed taking network performance as a main constraint and its targeted at security researchers.

It has been developed to easily allow developers build their custom plugins. Samples of those can be found in its codebase. In addition, more information about NINJA PingU can be found in the FAQ and dev pages.
